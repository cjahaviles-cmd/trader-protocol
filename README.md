# Trader Protocol — Deploy en Vercel

## Estructura del proyecto
```
trader-protocol/
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx
    └── App.jsx
```

## Pasos para subir a Vercel

1. Crea cuenta en github.com (si no tienes)
2. Crea cuenta en vercel.com con tu cuenta de GitHub
3. Sube esta carpeta a un repositorio de GitHub
4. En Vercel: New Project → importa el repo → Deploy
5. Tu app queda en una URL tipo: trader-protocol.vercel.app
